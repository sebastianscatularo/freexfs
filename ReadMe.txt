Directory Listing:

common  : contains files common to all VC projects.
dll : contains the .dll and .exe files in release mode
dllDebug : contains the .dll and .exe files in debug mode
forms : some sample forms.
lib : contains related .lib files 
Manager : a XFS manager in multiple thread mode. To be cmpatible with one coming with the SDK.
Samples/WosaXFSTest20081007 a sample test programs for XFS API and SPI
Samples/WosaXFSTest20100106 a sample test programs for XFS API and SPI

SPI : A sample SPI for windows printers.
XFS SPI : A SPI Wizard generator
PTR SPI : A PTR SPI wizard generator
PTROCX : an OCX to print forms

Files in dll subdirectory:
SPI_LQ2090.dll  :   a sample SPI for printers supporting Window device driver.
NI_XFSMgr.dll   :  a sample XFS manager dll
WosaXFSTest.exe  :  an APT testing program for XFS V3.0
WosaXFSTestV2.exe :  an APT testing program for XFS V2.0
xfs_conf.dll  : a supporting dll
SPIWizard.awx  : a SPI gengertor wizard for VC6. Please copy it to the template directory like 
		C:\Program Files\Microsoft Visual Studio\Common\MSDev98\Template
PTR SPI.awx   : a printer device SPI gengertor wizard for VC6. Please copy it to the template directory like 
		C:\Program Files\Microsoft Visual Studio\Common\MSDev98\Template


Registers:
XFS2.reg   Registers in HKEY_USERS
XFS_1.reg  Registers in HKEY_CURRENT_USER
